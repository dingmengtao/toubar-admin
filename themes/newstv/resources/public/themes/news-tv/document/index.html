Installation

Please follow below steps to setup Avatar component in your laravel project.

  • Install Laravel 5.x if you don't have exists project: https://github.com/laravel/laravel
  • Extract file laravel-avatar then copy folder Component to /app folder.
  • Add below code to psr-4 section of composer.json, then run composer dump-autoload to re-generate autoload files 
                                    
"WebEd CMS\\Avatar\\": "app/Components/Avatar/src"

    Then it will be look like:

                                    
"psr-4": {
    "App\\": "app/",
    "WebEd CMS\\Avatar\\": "app/Components/Avatar/src"
}
  • Open /config/app.php and add AvatarServiceProvider.php to provider section. 
                                    
'providers' => [
    ...
    // After others service providers
    WebEd CMS\Avatar\Providers\AvatarServiceProvider::class,
],
  • Create scaffold Laravel Auth if you don't create it yet. 
                                    
php artisan make:auth
  • Export public assets for Avatar component 
                                    
php artisan vendor:publish --provider="App\Components\Avatar\Providers\AvatarServiceProvider" --tag=assets
  • Open your application then go to http://{your-domain}/profile

Upgrade Guide

When you got a new version of WebEd CMS Laravel Avatar from Codecanyon. You just need to do below steps to upgrade it.

  • Extract file laravel-avatar then copy folder Component to /app folder.
  • Update psr-4 autoload section in composer.json 
                                    
"WebEd CMS\\Avatar\\": "app/Components/Avatar/src"
  • Update public assets for Avatar component. Namespace changed from App\Components\Avatar to WebEd CMS\Avatar 
                                    
php artisan vendor:publish --provider="WebEd CMS\Avatar\Providers\AvatarServiceProvider" --tag=assets --force

Dashboard

Index Page display videos simple statistics per month. and you can choose the year, the month and the day to display their statistics.

Categories

Categories Page display categories inserted by admin and you can add, edit and delete each category.

Menus

Menus Page display menus inserted by admin and you can add, edit and delete each menu. Also you can drag and drop menus to reorder them.

Posts & Videos

Posts Page Display published posts in addition to deleted posts and you can search for post(s). Also you can delete multi-posts

Pages

Pages Page Display custom content published by user such as (Homepage, About us, Contact us .. etc). You can Add, Edit and Delete pages.

Themes

Themes Page display themes inserted by admin and you can Active, Disable and Uninstall each theme.

Theme options

Theme options contains the Theme setting which allow you to choose the site color, items per page and Google site verification. Also you can choose the site template (Theme) from Themes page.

Configuration Settings

Configuration contains the configuration setting which allow you to choose config:

  • Basic: default homepage, main menu, site title, logo, favicon
  • Advanced: construction mode, Show admin bar
  • Google analytics: Google analytics script code
  • Captcha: Google captcha site key, Google captcha secret key
  • Social networks: Facebook, Youtube, Twitter, Google+, Instagram, Linkedin, Flickr
